## 网站信息

**博客标题**

网站的主标题，用于邮件通知模板、RSS 订阅、首页头部大标题等多个位置

**博客副标题**

网站的简短描述，用于首页标题后缀，同时呈现为网站 Logo

**博客描述**

用于 SEO 优化的 meta 描述，也用于 Open Graph 和 Twitter Card

**博客关键词**

SEO meta keywords 标签，用于 SEO 优化，多个关键词用逗号分隔

**网站 Favicon**

网站图标，显示在浏览器标签页

## 站长信息

**站长姓名**

显示在博客各处的站长名称标识，用于侧边栏作者卡片、关于页面个人介绍、页脚/文章版权信息等

**站长头像**

站长的头像图片，用于侧边栏作者卡片中圆形头像展示

## 备案信息

**ICP备案号**

ICP 备案号，显示在页脚备案信息区域，用于链接到工信部备案查询网站

**公安备案号**

公安备案号，显示在页脚备案信息区域，用于链接到公安备案查询网站

## 系统地址

以下地址在填写时需要包含 `https://` 前缀，并且无法以 `/` 结尾

**管理地址**

管理后台的完整访问地址，用于生成后台链接。邮件通知中，管理员相关通知（新评论、反馈投诉、友链申请）使用后台地址作为站点链接

**博客地址**

博客前端的完整访问地址，用于生成文章链接、RSS 订阅。邮件通知中，用户相关通知使用博客地址作为站点链接

## 扩展功能

**表情包配置**

博客评论中使用的表情包，JSON 配置文件的 URL

可使用 <https://fastly.jsdelivr.net/gh/talen8/emojis/emojis.json>

如需自定义可前往 <https://github.com/talen8/emojis> Fork 该仓库进行自定义修改

你就可以自定义为 `https://fastly.jsdelivr.net/gh/<用户名>/<仓库名>/emojis.json`

**Meting-API**

在线音乐解析服务地址，用于文章和动态中的音乐播放功能。默认使用 `https://meting.flec.top/api`，可替换为自建服务。

**头像服务**

用户头像服务 URL，`%s` 为邮箱 MD5 哈希占位符。默认使用 Cravatar：`https://cravatar.cn/avatar/%s?s=200&d=robohash`

**IP 查询**

IP 归属地查询服务 URL，`%s` 为 IP 地址占位符。默认使用 ip-api：`http://ip-api.com/json/%s?lang=zh-CN`

**封面制作**

封面制作功能的图片源 API 地址。默认使用 `https://pixhub.flec.top`

## 自定义代码

**自定义 Head**

插入到 `<head>` 标签的代码，支持 meta、link、script、style 标签

**自定义 Body**

插入到 `<body>` 标签开头的代码
