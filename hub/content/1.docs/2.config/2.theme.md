以下为默认主题的配置说明。

## 站点信息

**站长邮箱**

站长的联系邮箱，显示在博客各页面供用户联系，如政策协议页面、友链页面等

**站长简介**

站长的个人简介，显示在博客侧边栏作者卡片中

**主页地址**

博客首页的完整地址，在页脚版权信息和文章版权声明中，用于作者主页跳转

**建站日期**

网站成立的日期，用于计算建站时长

## 外观与主题

**背景图片**

博客的背景图，支持粘贴图片 URL 或上传，能够使用 302 重定向随机图

**侧边社交**

显示在博客侧边栏的个人社交链接，JSON 数组格式（图标使用 Remix Icon 图标库）

**页脚社交**

显示在页面底部的社交链接，JSON 数组格式（图标使用 Remix Icon 图标库）

**日间开始时间**

按时间段自动切换明暗主题的明亮主题开始时间

**夜间开始时间**

按时间段自动切换明暗主题的暗色主题开始时间，当与日间开始时间相同时，始终使用明亮主题

**字体配置**

自定义博客字体，格式：`字体文件URL|字体名称`（推荐使用千字网）

## 页面配置

**打字机文本**

首页轮播显示的文本列表，JSON 数组格式，循环显示文本

**首页布局**

博客端首页文章卡片区域的布局，支持双列瀑布流和单列卡片

**动态数量**

控制博客端动态页面的动态展示条数，默认为 30，不建议数量太多，容易导致页面卡顿

**公告**

用于展示公告内容，支持 HTML 标签和内联样式

*支持的标签：*`<br>`、`<p>`、`<strong>`、`<b>`、`<em>`、`<i>`、`<u>`、`<s>`、`<del>`、`<ins>`、`<mark>`、`<small>`、`<span>`、`<sub>`、`<sup>`、`<a href="..." target="...">`

*支持的样式：*`color`

**留言信**

博客端留言页面展开留言信封后展示的文本，建议 8-10 行

**友链描述**

在友链申请页面，用于生成站点信息配置中的 description 字段

**站点截图**

在友链申请页面，用于生成站点信息配置中的 screenshot 字段

**赞赏方式**

添加不同的赞赏方式，显示在文章底部，由赞赏名称和赞赏二维码组成

**公众号二维码**

上传微信公众号的二维码图片，用于在订阅页面展示，方便用户扫码关注公众号获取文章推送

**公众号名称**

微信公众号的名称，显示在二维码下方

**页脚链接**

显示在页面底部右侧的链接，JSON 数组格式

**站长形象**

站长的形象照片，在关于页面个人介绍右侧展示

## 关于页面

**个人描述**

关于页面顶部的主要描述文本

**描述提示**

在个人描述下方的补充性文本

**展览图片**

个人资料下方的装饰性图片

**个人资料**

用于博主信息的公开介绍，支持六个关键词，JSON 数组格式

**性格类型**

用于展示 16Personalities，预设全部类型可快速选择

**座右铭**

展示两行文本组成的座右铭，JSON 数组格式

**一言**

在座右铭下方的补充性文本

**联系方式**

关于页面的联系方式列表，JSON 数组格式

**创作平台**

博主的其它创作平台列表，JSON 数组格式

**版本信息**

此网站技术栈版本信息，JSON 数组格式

**站长联盟**

加入的联盟/组织列表，JSON 数组格式

**心路历程**

关于本站的介绍和成长故事

## 菜单管理

每个主题在 `theme.json` 中定义菜单结构，管理端通过「主题配置」页面进行可视化编辑。

### 菜单类型

系统支持三种菜单类型，对应不同的前台显示位置：

| 类型 | 显示位置 | 说明 |
| --- | --- | --- |
| 聚合菜单 | 首页右上角 | 点击展开的网格布局菜单 |
| 导航菜单 | 页面顶部 | 主导航栏菜单，支持下拉子菜单 |
| 页脚菜单 | 页面底部 | 页脚链接区域 |

### 菜单字段

每个菜单项支持以下字段：

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| 标题 | 输入框 | 是 | 菜单显示的名称 |
| 链接 | 输入框 | 是 | 菜单跳转的 URL |
| 图标 | 输入框 | 否 | Remix Icon 类名或图片 URL |
| 排序 | 数字 | 否 | 数值越小排序越靠前 |
| 启用 | 开关 | 否 | 是否在前台显示 |

### 链接类型

| 类型 | 说明 | 示例 |
| --- | --- | --- |
| 内部链接 | 指向博客内部的页面 | `/`、`/friend`、`/about` |
| 外部链接 | 指向外部网站的 URL | `https://github.com` |
| 空链接 | 不跳转，用于分组标题 | `javascript:void(0)` |

### 图标说明

菜单支持两种图标：

| 图标类型 | 判断方式 | 示例 |
| --- | --- | --- |
| Remix Icon | 非 URL 格式 | `ri-home-line` |
| 图片图标 | `http://`、`https://`、`/` 开头 | `https://example.com/icon.png` |
